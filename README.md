# MonaLab الأكاديمي

**منظومة أكاديمية عربية** لمساعدة الطلاب والباحثين على إعداد بحوث وتقارير جامعية بالعربية، بهيكل علمي واضح وقابل للتعديل والتصدير، خلال دقائق.

---

## وصف الموقع

MonaLab منصة ويب تجريبية مصممة خصيصاً لاحتياجات الطالب والباحث العربي. توفّر واجهة عربية بالكامل يتفاعل من خلالها المستخدم مع **«الأكاديمي الذكي»** — مساعد يعتمد على الذكاء الاصطناعي (Gemini أو NVIDIA) لتوليد بحوث وتقارير أكاديمية منسّقة.

- **إنجاز البحوث والتقارير الجامعية بالعربية باحترافية خلال دقائق.**
- بحوث منسّقة بهيكل علمي واضح: مقدمة، فصول، خاتمة، مراجع مقترحة.
- تصدير مباشر إلى **Word** و**PDF** جاهز للطباعة أو الإرسال للمشرف.
- إمكانية حفظ أكثر من **جلسة بحث** والعودة إليها في أي وقت.
- تحكم كامل بالمحتوى: توسيع فصل، إعادة صياغة، تحسين لغة، مع تذكير دائم بأهمية المراجعة والتوثيق.

المنظومة لا تهدف إلى استبدال الجهد البحثي الحقيقي، بل إلى تمكين المستخدم من التركيز على الفكرة العلمية بينما تتكفّل الأدوات بتنظيم المحتوى وصياغته الأولية بأسلوب أكاديمي رصين.

---

## الأهداف

- **سد فجوة أدوات البحث والكتابة الأكاديمية باللغة العربية** وتمكين الطالب والباحث من إنتاج مسودات منظمة بسرعة.
- **تبسيط رحلة البحث** من الفكرة الأولية إلى بحث منسق جاهز للطباعة، دون المساس بالمعايير العلمية.
- **توفير تجربة استخدام مركزة** حول سيناريو واحد واضح: إعداد بحث أو تقرير أكاديمي قابل للتعديل والتطوير.
- **الالتزام بالأمانة العلمية** وتشجيع التوثيق الصحيح للمصادر، مع وضوح حدود الذكاء الاصطناعي وأهمية المراجعة النقدية.

---

## الرؤية

- أن تصبح MonaLab **مرجعاً عربياً** لمساندة المجتمع الأكاديمي في إنتاج محتوى علمي عالي الجودة ومنظم.
- تصميم المنظومة لخدمة **مختلف التخصصات** مع احترام التباينات بين العلوم الإنسانية والعلوم التطبيقية.
- إتاحة **مساحة كبيرة للتخصيص** وفق متطلبات كل مؤسسة تعليمية أو مشرف أكاديمي.
- الحرص على **خصوصية المستخدم** وعدم استخدام بياناته في أي تدريب إضافي، مع إمكانية دعم خيارات تخزين سحابي لاحقاً بشكل واضح وشفاف.

---

## لمن صُمّمت المنظومة؟

- المشرفون والمدرّسون الذين يرغبون في مساعدة طلابهم على تنظيم أعمالهم.

## التقنيات المستخدمة

| التقنية | الاستخدام |
|--------|-----------|
| **Next.js 15** | إطار العمل والـ API Routes |
| **React 18** | الواجهة الأمامية |
| **Tailwind CSS** | التنسيق والتصميم |
| **Prisma** | قاعدة البيانات (اختياري، لحفظ المحادثات) |
| **Gemini / NVIDIA** | نماذج الذكاء الاصطناعي للتوليد النصي |

يدعم المساعد **الرد التدريجي (Streaming)** عند استخدام Gemini لظهور الإجابة فوراً بشكل تدريجي، مع تلميح للسرعة حسب طول الطلب (رسائل قصيرة → رد سريع، طلبات بحث طويلة → قد يستغرق دقيقة أو أكثر).

## المتغيرات البيئية

أنشئ ملف **`.env.local`** في جذر المشروع وضَع فيه أحد الخيارين (أو كليهما):

| المتغير | الوصف |
|---------|--------|
| `GEMINI_API_KEY` أو `GOOGLE_API_KEY` | مفتاح واجهة Gemini من Google AI — يُستخدم أولاً مع دعم الـ Streaming. |
| `Nivedi_api_key` | مفتاح NVIDIA API — يُستخدم كبديل عند غياب Gemini أو فشله (مثل 429). |

للحفظ الاختياري للمحادثات في قاعدة بيانات:

- `DATABASE_URL` و `DIRECT_URL` (اتصال Prisma بـ PostgreSQL، مثلاً عبر Supabase).

ملف **`.env`** و **`.env.local`** غير مرفوعين في المستودع لأسباب أمنية؛ انسخ المتغيرات يدوياً أو استخدم `.env.example` كمرجع إن وُجد.

## هيكل الصفحات الرئيسية

- **الرئيسية** — صفحة هبوط تعرّف بالمنظومة وتربط ببدء بحث جديد.
- **المحادثة (الأكاديمي الذكي)** — واجهة الدردشة مع المساعد، جلسات متعددة، تصدير Word/PDF.
- **كيف يعمل** — دليل آلية العمل (جمع المعطيات → توليد المسودة → المراجعة والتصدير).
- **من نحن** — الرؤية، الرسالة، المبادئ، والمطوّر.
- **سياسة الخصوصية / شروط الاستخدام / التنبيهات القانونية** — صفحات قانونية وإعلامية.
